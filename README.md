# 🎲 Random Message Generator

A simple JavaScript project that generates a random message each time the program runs or the page is refreshed.

This project was created as part of a learning challenge to practice JavaScript, Git, and working with the command line.

---

## 📌 Project Description

The application randomly generates a message composed of multiple parts.  
Each time the program runs, the user receives a different message.

The messages are built from at least **three separate data sources** to ensure variety and randomness.

Examples of generated messages may include:

- Inspirational quotes
- Funny phrases
- Random motivational advice

---

## 🎯 Project Objectives

- Build a **random message generator** using JavaScript
- Practice **Git version control**
- Work with the **command line**
- Develop the project **locally**

## ⚙️ How It Works

The application:

1. Stores different parts of messages in JavaScript arrays.
2. Uses `Math.random()` to select random elements.
3. Combines them into a final message.
4. Displays the message on the webpage.
